# HR Analytics Dashboard

## Project Overview
This project analyzes HR data to understand employee attrition, workforce trends, and key HR performance indicators.

## Tools Used
- Power BI
- SQL
- Excel
- Tableau

## Key Insights
- Employee Attrition Analysis
- Department-wise Employee Distribution
- Job Role Analysis
- Gender and Age Demographics
- Employee Performance Metrics

## Files Included
- Power BI Dashboard (.pbix)
- SQL Queries (.sql)
- Excel Dataset (.xlsm)
- Tableau Workbook (.twbx)
- Project Presentation (.pptx)

## Purpose
This project demonstrates data analysis and visualization skills using real-world HR data.
